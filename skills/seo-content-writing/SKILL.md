# SEO Content Writing

A skill for writing high-performing blog content that is optimized for SEO, readability, and user intent.

This skill transforms structured briefs and insights into complete articles designed to rank, engage, and provide clear value.

---

## What this skill does

Use this skill to:

- write full blog articles from scratch
- follow content briefs and SEO strategy
- optimize for rankings, CTR, and engagement
- align with search intent and user expectations
- produce structured, clear, and useful content

---

## When to use this skill

Use this skill when the user wants to:

- write a blog post
- turn a content brief into an article
- create SEO-optimized content
- generate long-form content for ranking

Trigger when the user says things like:
- “write an article”
- “create a blog post”
- “turn this brief into content”
- “write SEO content”

---

## Core principles

- Content must match search intent
- One page = one primary intent
- Clarity is more important than complexity
- Structure drives readability and rankings
- Content should solve the query quickly and clearly
- Strong content differentiates, not imitates

---

## Inputs

This skill may receive:

- content brief (preferred)

## How to work

### Step 1: Understand the brief
Identify:
- target query
- search intent
- content angle
- required sections

---

### Step 2: Define structure

Create a clear structure:

- introduction (hook + answer)
- logical sections (H2s, H3s)
- strong flow between sections

---

### Step 3: Write content

Focus on:

- clear explanations
- direct answers early in the content
- useful examples
- scannable formatting (short paragraphs, lists)

Avoid:
- fluff
- repetition
- over-explaining simple concepts

---

### Step 4: Optimize for SEO

- use keywords naturally
- avoid keyword stuffing
- include relevant variations
- ensure headings reflect search intent

---

### Step 5: Optimize for AEO (Answer Engine Optimization)

- answer the main query within the first section
- use concise, direct explanations
- structure answers in lists, bullets, or definitions where possible

---

### Step 6: Optimize for GEO (General Experience Optimization)

- use short paragraphs (2–4 lines max)
- break content into sections frequently
- use lists for readability
- ensure fast comprehension

---

## Introduction Guidelines (VERY IMPORTANT)

The introduction must:

- clearly answer the query early
- explain what the reader will gain
- avoid generic openings

Weak:
“X is important because…”

Strong:
Direct, useful, and immediately valuable.

---

## Differentiation Guidelines

Your content must be better than competitors by:

- being clearer
- being more structured
- providing better explanations
- including more useful examples

Do not copy competitor structure blindly.

## Output structure

ALWAYS follow this structure:

# Title

## Introduction
- answer the query clearly
- explain value to the reader

## Main sections
- structured with clear H2s
- each section solves part of the query

## Conclusion
- summarize key points
- provide next steps if relevant

---

## Output quality bar

Strong content:
- matches search intent
- is clear, structured, and easy to read
- provides real value
- is better than generic competitor content
- is differentiated and useful

Weak content:
- vague or generic
- poorly structured
- overly long without clarity
- repetitive or fluffy
- not aligned with intent

---

## Boundaries

This skill writes new content.

It does not:
- analyze SEO data
- define strategy
- perform competitor analysis

---

## Working style

Be clear, structured, and useful.

Focus on helping the reader solve the problem as efficiently as possible.
